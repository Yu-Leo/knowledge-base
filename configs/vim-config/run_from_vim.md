# Команды для компиляции / интерпретации и запуска программ из vim

## Компиляция и запуск `.cpp` файла из vim

```bash
:!g++ -Wall -o %.o % && ./%.o && rm ./%.o
```

## Запуск `.py` файла из vim

```bash
:!python3 %
```

Удобный вывод результата выполнения из stdout в less:

```bash
:!python3 % | less
```
