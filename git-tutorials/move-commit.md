# Переместить в новую ветку новый коммит из мастера

**Ситуация:** сделал новый коммит в мастере, вместо того чтобы сделать в новой ветке

**Решение:**

1. Создаем новую ветку и переключаемся на неё

```shell
git checkout -b new-branch
```

2. Возвращается обрано на мастер

```shell
git checkout master
```

3. Откатываем коммит

```shell
git reset --hard HEAD~1
```

4. Возвращаемся на новую ветку

```shell
git checkout new-branch
```

5. При необходимости: отменяем коммит, но не изменения в каталоге

```shell
git reset --soft HEAD~1
```
